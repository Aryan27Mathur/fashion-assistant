## Installation

### Loading the Extension in Chrome

1. **Open Chrome Extensions Page**
   - Open Google Chrome
   - Navigate to `chrome://extensions/` in the address bar
   - Alternatively, go to Chrome menu (three dots) → Extensions → Manage extensions

2. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner of the extensions page

3. **Load the Unpacked Extension**
   - Click the "Load unpacked" button that appears after enabling Developer mode
   - Navigate to and select the `fashion-assistant` folder (the folder containing `manifest.json`)
   - Click "Select Folder" or "Open"

4. **Verify Installation**
   - The extension should now appear in your extensions list
   - You should see the extension icon (shirt.png) in your Chrome toolbar

## Usage

### Accessing the Extension

1. **Open the Side Panel**
   - Click the extension icon in the Chrome toolbar
   - The Fashion Assistant side panel will open on the right side of your browser

2. **Capturing Images**
   - Navigate to any webpage with fashion items or images
   - Click the extension icon to open the side panel
   - The extension will capture images from the current page
   - Captured images will appear in the side panel

3. **Finding Similar Items**
   - After capturing an image, click the "Find Similar Items" button
   - The extension will analyze the image and provide recommendations
   - Similar items and recommendations will be displayed in the side panel

4. **Downloading Images**
   - Use the "Download Image" button to save captured images to your device

5. **Starting a New Capture**
   - Click "New Capture" to clear the current image and start fresh

### Features

- **Image Capture**: Automatically captures images from web pages
- **Similar Item Search**: Uses AI to find similar fashion items
- **Side Panel Interface**: Clean, modern interface accessible from any webpage
- **Image Download**: Save captured images for later reference
